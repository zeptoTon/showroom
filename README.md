# Objective

This project is demonstrating full stack technology by using a showroom

## Technology Used:
- Typescript
- Mobx
- React
- Jest Test Runner
- Express
- Webpack
- tslint
- Mongoose
- Promise & async/await
- pm2

## TODO:
- SASS
- enzyme, sinon in Frontend Testing

## Setup
`yarn install`

`npm install -g pm2`

## Start Dev
`yarn start`

`pm2 start process.json`

## Test
`yarn test`

## Config
package.json
- MONGODB, connection string for jest test env.

process.json
- MONGODB, connection string for dev env.
